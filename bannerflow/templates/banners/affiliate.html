Configuración de Afiliado

Configura tus links de afiliado para que el generador los limpie automáticamente y extraiga sólo la URL real del producto.

Adidas (Awin)

El link de afiliado Awin se encuentra al principio del enlace. Copia la parte del link que aparece antes de la URL del producto.
Ejemplo completo:
https://www.awin1.com/cread.php?awinmid=79922&awinaffid=1674245&ued=https%3A%2F%2Fwww.adidas.cl%2F...
Prefijo a guardar:
https://www.awin1.com/cread.php?awinmid=79922&awinaffid=1674245&ued=https%3A%2F%2F

Déjalo vacío si no usas links de afiliado Awin.
Sodimac

El link de afiliado Sodimac se añade al final del enlace. Copia el inicio de la parte afiliada (el separador que marca dónde empieza).
Ejemplo completo:
https://www.sodimac.cl/.../121227892?eid=666_campaign=trends_usr=9iw92e
Sufijo a guardar: ?eid=

Normalmente es ?eid=. Déjalo vacío si no usas links de afiliado Sodimac.
Cancelar
¿Cómo funciona?
  • Al pegar un link en el generador, el sistema detecta automáticamente si tiene una parte de afiliado.
  • Si el link coincide con tu configuración, se limpia y se usa la URL real del producto para extraer sus datos.
  • Si el link tiene una parte de afiliado que no coincide con lo que guardaste aquí, verás una advertencia (el producto se extrae de todas formas).
  • Si no tienes links de afiliado, puedes dejar ambos campos vacíos.
